# Changelog

## Version 1.1
- Features: Added comprehensive management of display settings including brightness, gamma, resolution, refresh rate, DPI, scale, rotation, monitor state, HDR, and screen backlight control (hardware dependent).
- Usability: Implemented monitor selection column so each control row applies to the selected monitor.
- Usability: Field-based adjustments are applied by pressing Enter in numeric fields for precision.
- Safety: Introduced confirmation dialogs on display changes with automatic 10-second revert to prevent misconfiguration.
- Improvements: Initial resolution and refresh rate selection on program start automatically reflects current active display settings.
- UI: Added additional controls for HDR and monitor power state toggling.
- Code Quality: Refined signal handling and modularized resolution and refresh rate update logic.
- Documentation: Updated About dialog to reflect contribution by ChatGPT, Perplexity, and Claude.
